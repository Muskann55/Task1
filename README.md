# QSkill Frontend

A modern React-based web application that provides AI-powered text generation and language translation features through a clean and responsive user interface.

## Features

### Text Generator
- Generate text based on user prompts
- Fast and responsive interface
- Easy-to-use input fields
- Real-time output display

### Translator
- Translate text between multiple languages
- User-friendly translation interface
- Instant translation results

### UI Features
- Modern React frontend
- Responsive design
- Clean navigation
- Component-based architecture

## Installation

### Clone Repository

```bash
git clone <repository-url>
```

### Navigate to Project

```bash
cd qskill-frontend
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Application will run at:

```text
http://localhost:5173
```

---

## Build for Production

```bash
npm run build
```
